Concepto y breve historia de CSS Lectura 3


En primer lugar tenemos que identificar sus siglas,  C cascading S style S sheets lo 
que traduce Hojas de estilo en cascada.  Esta herramienta es utilizada por los 
navegadores y programadores web para el diseño de las páginas o sitios de internet. 
En un principio el aspecto visual dependía de los atributos que se le aplicasen a las 
etiquetas de HTML, pero con el pasar del tiempo y la evolución del lenguaje se vio la
necesidad de separar el contenido semántico del aspecto visual. Es entonces donde 
comienza la historia de CSS:
- 1994 y 1995: Hakon Wium Lie y Bert Bos unen esfuerzo para definir el lenguaje CSS
- 1996: W3C publicó la primera recomendación oficial llamada CSS nivel 1
- 1997: W3C crea un equipó exclusivo para el desarrollo de CSS
- 1998: Se publica la segunda recomendación oficial conocida como CSS nivel 2
- Actualidad, aunque los progresos en los diferentes lenguajes son robustos, cabe 
  aclarar que la recomendación oficial de W3C para el 	CSS3 aún no llega debido a 
  que los navegadores no poseen todavía el soporte completo para el nivel 2.1, se 
  espera que el lenguaje se siga desarrollando a la par de los navegadores.
