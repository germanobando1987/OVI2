Consejos para codificar Lectura 5


Buenas practicas a la hora programar
Hace algunos años decidí internarme en el mundo de la programación, fue casualidad o
la busqueda de un nuevo campo laboral lo que me llevo al campo de la 
codificación y me encontré con algoritmos, sintaxis, lenguajes y muchos elementos 
más, que en un principio desbordaron mi capacidad de retener y comprender que hacia 
cada uno de esos ítems; entonces pensé que la mejor manera comprender y ser eficaz 
a la hora de desarrollar un código para el desarrollo de software era tener  unas 
buenas costumbres a la hora de programar. 

Por ende quiero compartir estos 5 consejos para ser eficiente en la codificación:

- Realiza una sintaxis clara y ordenada: la identación es esencial para visualizar 
  un orden a tu código
- Realiza comentarios: los comentarios de las secciones de código que ejecutan 
  acciones relevantes son básicos para que no pierdas tiempo en buscar algo necesario
  para cambios
- No repitas código: es buena práctica releer tu código para asegurarte que no has 
  repito código innecesario, así podrás factorizar lo que se pueda para volver más 
  eficiente el código.
- Realiza módulos: es recomendable separar el código en módulos de funciones 
  concretas para no tener confusiones ni enredos a la hora de realizar algún cambio
- Copias de seguridad: es importante realizar una o varias copias de seguridad de tu 
  código por si en algún momento la información se extravía o daña. 

Podrás encontrar en el web muchos mas consejos para la buena práctica a la hora de 
codificar.
