Definición y breve historia Lectura 1


HTML (Hyper Text Markup Language) o lenguaje de marcas 
de hipertexto es esencial para la construcción de una 
página web o sitio web. En este lenguaje es fundamental 
el correcto posicionamiento de la estructura que se 
llevara en el sitio por ende es fundamental antes de 
empezar a generar un código, diseñar tanto el contenido 
como los espacios a utilizar en el portal.
Este lenguaje funciona a partir de texto, esto quiere 
decir que los archivos que se presentan en el sitio no
están incrustados en el código, se posicionan fuera de 
él y se los invoca para que el navegador web interprete 
la instrucción y realice la acción necesaria, de esta 
manera los videos, imágenes audio o cualquier contenido 
multimedia que pretendamos mostrar en un sitio serán 
almacenados fuera del código. 
En cuanto a la historia de este lenguaje podemos 
establecer puntos clave de ella:
- 1980: origen en el CERN (organización europea para 
la investigación nuclear) por el físico Tim Berners-Lee,
quien propuso un nuevo lenguaje de hipertexto 
(se debe mencionar que ya existían este tipo de lenguajes)
- 1991: publicación de “HTML Tags” primer documento 
  descriptivo de HTML realizado por Tim Berners-Lee y el 
  ingeniero de sistemas Robert Cailliau.
- 1993: Se realiza la primera propuesta oficial para 
  establecer HTML como estándar
- 1995: Se genera HTML 2.0 y es catalogado como estándar 
  oficial 
- 1997: Se publica la versión HTML 3.2 el 14 de enero  
- 1998: El 24 de abril de 1998 se publica HTML 4.0 y el 
  avance es significativo, entre sus mejoras esta la 
  implementación de hojas de 	estilo CSS
- 1999 a 2004: ciertos organismos dejan de lado HTML y 
  centran esfuerzos en implementar XHTML, por tanto 
  empresas como Apple, Mozilla y Opera trabajan en 
  conjunto para estandarizar HTML5
- 2008: Se publica el primer borrador de HTML5
